Model Training
Precision
This is a performance metric used to evaluate the accuracy of positive predictions made by a classification model. It is the ratio of true positives (TP) to the sum of true positives and false positives (FP). A high precision score indicate that the model makes few false positive predictions relative to the true positive predictions, making it highly reliable when it predicts positive instances. A low precision on the other hand shows that the model has a high rate of false positives, making the predictions less trustworthy
 
3.4.2	Recall
This is a performance metric used to evaluate the ability of a classification model to accurately identify all positive instances from the actual positive class. It is calculated as the ratio of true positives (TP) to the sum of true positives and false negatives (FN).
 
A high recall score suggests that the model effectively captures a large proportion of positive instances, minimizing the number of false negatives. A low recall on the other hand suggests that the model misses many positive instances, leading to a high rate of false negatives
3.4.3	Accuracy
Accuracy is a performance metric used to evaluate the overall correctness of predictions made by a classification model. It is calculated as the ratio of the number of correctly predicted instances (true positives and true negatives) to the total number of instances.
 
A high accuracy score indicates that the model makes a large proportion of correct predictions, while a low accuracy score suggests that the model makes significant number of incorrect predictions, indicating poor performance.
3.4.4	F1 Score
The f1 score is a single metric that combines both precision and recall, providing a balanced assessment of a classification model’s performance.
 
A high F1 score indicates that the model has high precision and recall, indicating that it makes accurate positive predictions while capturing a large proportion of positive instances
